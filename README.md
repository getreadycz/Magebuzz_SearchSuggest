# Search Suggest and Autocomplete
> Search Suggest and Autocomplete is an extension which gives your customers more flexibility while searching any product in a fast, easy and effective way. 


* Allow to manage product attributes to search for
* Return suggested search result using Ajax


## Features
   Overview

Search Suggest and Autocomplete extends Magento search and helps it run better for your online store. Customers can easily search products without loading the page. This extension uses Ajax to return match items with search query quickly. Moreover, the module configuration allows you to change the look and feel as well as customize search query to meet your requirements.


   Core Features

* Search is based on Ajax, quickly display match items
* Allow admin to configure product attributes included in search query
* Searches start after the minimum value of character has been entered
* Enable to show/hide out of stock products
* Allow to limit items displayed in the dropdown list
* Allow to show product pictures in the search results
* Ability to customize the dimension of product thumbnail image
* Enable to display product description in the search results
* Default open search result link in a new window
* Allow to edit the heading of the search result list
* Default search pre-loader & complete icon
* Allow customers to come directly to their desired product pages by clicking on the product link
* Allow to sort search results in alphabetical order
